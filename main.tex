\section{Combining lhs2TeX and ghci4luatex}


Let |sqr| be the square function:

\begin{code}
sqr n = n * n
\end{code}

\begin{ghci}
  sqr :: Num a => a -> a
  sqr n = n * n
\end{ghci}

Then  |sqr 3| is equal to \hask{sqr 3 :: Int}.
